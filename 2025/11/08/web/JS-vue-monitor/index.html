多视角监控回放页面实现

背景

在实际工作中，我们经常需要对监控视频进行回放，但是通常只有一个视角的视频，无法满足实际需求。因此，我们需要实现一个多视角监控回放页面，支持同时播放多个视角的视频。

需求

  • 支持同时播放多个视角的视频
  • 支持视频的暂停、播放操作
  • 支持视频的倍速播放
  • 支持视频的拖动播放
  • 支持视频的音量调节
  • 支持时间轴标记
  • 支持视频的截图
  • 支持视频的全屏播放

技术栈

  • Vue 3
  • TypeScript
  • flv.js
  • TailwindCSS

视频播放器

时间轴

主页面

难点

  • 多视频同步播放
  • 播放指定片段的指定时间点
  • 视频切换时，保持播放进度
  • 展示当前播放时间点

总结

通过 Vue 3 + TypeScript + flv.js + TailwindCSS 技术栈，我们成功实现了一个功能完善的多视角监控页面。但是由于flv.js对H.265的支持不是很好，改为mpegts.js能更高效处理 H.264、H.265（HEVC）视频编码和 AAC 音频编码，适配高清甚至 4K 监控场景.不过这两都是靠GPU来解码H.265视频,遇到一些比较老旧的电脑,可能出现无法播放的情况.要适配更多的电脑,需要jessibuca等播放器来解码H.265视频.
